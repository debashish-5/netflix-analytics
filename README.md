# skillytixs-task-1

A data science project focused on analyzing Netflix content using Python and Jupyter Notebooks.

## Features
* **Exploratory Data Analysis (EDA)**: Jupyter Notebook for identifying trends, patterns, and insights within Netflix's movie and show database.
* **Dataset**: Includes `mymoviedb.csv`, a structured dataset containing information about movies and shows.

## Getting Started
1. Clone this repository to your local machine.
2. Open the `Netflix data analysis.ipynb` file in **Jupyter Notebook** or **JupyterLab**.
3. Ensure you have the necessary libraries installed:
   ```bash
   pip install pandas matplotlib seaborn
